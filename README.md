# Three GLUT

A lightweight scene editor built with React, TypeScript, Vite, and Three.js.

## Features

- VS Code-style editor layout
- 2D and 3D scene editing
- Undo and redo history
- Grab mode with axis locking
- Scene explorer with add, duplicate, rename, and delete actions
- JSON import/export
- GLUT C++ export

## Getting Started

Install dependencies:

```bash
npm install
```

Run the app locally:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

Run lint checks:

```bash
npm run lint
```

## Notes

- `dist/` and `node_modules/` are not committed to GitHub.
- The editor is designed to work on desktop and smaller screens.
